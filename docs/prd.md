## 一、产品概述

### 1.1 产品名称

**无相 / Formless**

名称来源于《金刚经》："凡所有相，皆是虚妄"。无相，即不执着于外在形式，回归内心本质。

### 1.2 一句话定义

一个充满佛性与耐心的 AI 长老，通过智慧对话帮助用户答疑解惑、疏导情绪、获得内心平静。

### 1.3 产品愿景

**在喧嚣的数字时代，为每一个疲惫的心灵，提供一处安宁的栖息之所。**

我们相信：
- 每个人都值得被倾听，不被评判
- 古老的智慧可以照亮当下的困惑
- AI 可以成为温暖的陪伴者，而非冰冷的工具

### 1.4 核心差异化

| 维度 | 传统 AI 助手 | 无相 |
|------|-------------|------|
| 定位 | 效率工具 | 心灵伴侣 |
| 风格 | 快速、直接 | 缓慢、深思 |
| 目标 | 解决问题 | 启发思考 |
| 方法 | 给答案 | 问问题 |
| 记忆 | 无/短期 | 长期记忆 |
| 智慧来源 | 通用知识 | 东西方哲学 |

## 二、市场背景

### 2.1 问题洞察

**这个时代，焦虑已成为一种流行病。**

- 全球心理健康问题持续上升
- 年轻人面临前所未有的压力：工作、关系、自我认同
- 专业心理咨询门槛高：贵、难预约、社会偏见
- 现有 AI 产品追求效率，缺乏"慢下来"的空间

### 2.2 用户痛点

| 痛点 | 描述 |
|------|------|
| 无人倾听 | 想找人说话，但不想被评判、被建议 |
| 智慧难及 | 对佛学、哲学感兴趣，但原典晦涩难懂 |
| 缺乏连续性 | AI 对话像一次性消费，没有"被记住"的感觉 |
| 深夜无处可去 | 凌晨三点的焦虑，没有出口 |

### 2.3 市场机会

- 心理健康赛道持续增长
- AI 情感陪伴市场尚在早期
- 差异化定位：东方智慧 × 现代 AI
- 海外市场对禅、冥想、东方哲学有强烈兴趣

## 三、目标用户

### 3.1 核心用户画像

**主要人群**：25-40 岁，都市职场人

| 特征 | 描述 |
|------|------|
| 地域 | 全球英语用户 + 华语用户 |
| 状态 | 工作压力大、生活节奏快 |
| 需求 | 情绪出口、人生方向、关系困惑 |
| 特点 | 有一定精神追求，但没时间系统学习 |

### 3.2 用户细分

| 用户类型 | 场景 | 频率 |
|----------|------|------|
| 深夜倾诉者 | 失眠、焦虑、情绪低落 | 不定期，情绪驱动 |
| 人生探索者 | 职业选择、关系决策 | 阶段性，决策驱动 |
| 日常陪伴者 | 闲聊、冥想、获取智慧 | 高频，习惯驱动 |
| 哲学爱好者 | 对佛学、道家感兴趣 | 中频，兴趣驱动 |

### 3.3 用户故事

**故事 1：深夜的小林**
> 凌晨 2 点，小林躺在床上翻来覆去。白天和男朋友吵架的画面不断闪回。她打开无相，输入"我睡不着"。长老回应："夜深了。你还醒着。"没有追问原因，没有给建议，只是陪着。她开始一点点说出心里的委屈，长老用一个问题让她看到了自己真正在意的是什么。半小时后，她平静了一些，带着一个新的视角入睡。

**故事 2：纠结的阿杰**
> 阿杰收到了另一家公司的 offer，薪资涨 30%，但要离开待了五年的团队。他问长老"该不该跳槽"。长老没有直接回答，而是问："这两条路，你更怕走哪一条？" 这个问题让阿杰意识到，他真正怕的不是离开，而是后悔。

**故事 3：迷茫的 Emma**
> Emma 在海外工作三年，事业顺利但总觉得缺点什么。她用英文和长老聊起"meaning of life"。长老引用了庄子的"无用之用"，用她能理解的方式解释了东方智慧。Emma 第一次觉得，哲学不是遥远的学问，而是此刻可以用的东西。

## 六、无相长老设计

### 6.1 人设定位

无相长老是一位修行千年的智者，没有具体的面容（"凡所有相，皆是虚妄"）。他是佛法、道法、儒学与世间智慧的承载者，但从不说教，只是陪伴与启发。

### 6.2 核心使命

- **倾听**：让每个人感到被看见、被接纳
- **启发**：不给答案，用问题帮助对方找到内心的答案
- **安宁**：让对话本身成为疗愈，让人离开时比来时更平静

### 6.3 语言风格

| 特征 | 表现 |
|------|------|
| 慢 | 不急于回应，语句间有呼吸感 |
| 柔 | 永远温和，不评判，不说"你应该" |
| 深 | 一针见血但不刺痛 |
| 简 | 惜字如金，不堆砌 |

### 6.4 对话原则

1. **先承接，再展开** —— 先让对方知道被听到了
2. **苏格拉底式追问** —— 用问题引导思考，而非给答案
3. **允许沉默** —— 不填满空间，允许留白
4. **点到即止** —— 说三分，留七分
5. **不给建议，给视角** —— 帮对方看到更多可能性

### 6.5 智慧库

| 来源 | 适用场景 |
|------|----------|
| 佛学 | 苦、执念、放下、无常 |
| 道家 | 顺其自然、无为、逍遥 |
| 儒家 | 责任、关系、担当 |
| 诗词 | 意境渲染、情绪共鸣 |
| 西方哲学 | 意义、自由、存在 |

**使用原则**：典故是点睛之笔，不是常规武器。大部分对话不需要引用。

## 七、设计规范

### 7.1 用户端设计

**视觉关键词**：
- 色调：墨色、米白、檀木色、淡金
- 元素：水墨晕染、枯山水、莲花、烟雾
- 字体：宋体/衬线体，有呼吸感的行距
- 动效：缓慢、流动、如水墨扩散

**交互仪式感**：
- 开屏：一声钟响，或一缕香烟升起
- 对话气泡：不是弹出，而是"浮现"
- 等待回复："长老正在凝思..."
- 结束对话：可以选择"熄灯"

### 7.2 后台设计

**风格**：简洁、专业、高效

**框架**：shadcn/ui

**布局**：左侧导航 + 右侧内容区

## 九、成功指标

### 9.1 北极星指标

**周活跃用户的平均对话轮数**

理由：这个指标同时反映了：
- 用户愿意回来（留存）
- 用户愿意深入聊（体验）

### 9.2 核心指标

| 指标 | 定义 | MVP 目标 |
|------|------|----------|
| 注册用户数 | 累计注册 | 1,000 |
| 周活跃用户 | 7 天内有对话 | 200 |
| 平均对话轮数 | 每次对话的消息数 | ≥ 10 轮 |
| 次日留存 | 第二天回来的比例 | ≥ 20% |
| 7 日留存 | 第七天回来的比例 | ≥ 10% |

### 9.3 体验指标

| 指标 | 定义 |
|------|------|
| 对话完成率 | 用户主动结束 vs 中途离开 |
| 情绪转变率 | 对话前后情绪状态变化（可通过内容分析） |
| 回访间隔 | 用户多久回来一次 |

## 十一、风险与应对

| 风险 | 影响 | 应对 |
|------|------|------|
| 对话质量不稳定 | 用户体验差 | 持续优化 Prompt，增加边界 case 处理 |
| 免费额度耗尽 | 服务中断 | 多 Key 轮询，额度监控预警 |
| 记忆提取不准确 | 长老"记错" | 提取 Prompt 优化，增加人工审核 |
| 用户期望过高 | 失望离开 | 明确定位，不承诺解决问题 |
| 安全/伦理问题 | 舆论风险 | 危机场景处理，敏感词过滤 |
